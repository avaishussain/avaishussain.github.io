GUI Cleaned up and made separate classes for each render. Performance has gone up by a lot since the last meeting as the switch to LIBGDX was successful. Working on Saving and loading, still needs a lot of work. Work on different animations for characters.

1/03/19

Combat Animation discussed, need to work on getting the merchants working, purchasing weapons from shops and adding them to inventory. Few bugs found in system which need to be fixed, Memory usage on pause, Player spawns over object layer, Projectile animation is overly aggressive.

6/03/19

Different Theme maps are needed, Snow, Grass, Fire, Dessert etc and plus the rooms for different styled maps. Give enemies attributes, hp, damage etc. issue with projectile and animation fixed and updated into the master. Should make different types of quests. Pathfinding still a major issue.

08/03/19

New Quest Type completed, Character animation completed too, game tracks NPCs and Enemies. Tidied up code, Sprite manager for player animations. Trading between NPCs and player needs to be implemented. Game Screen class needs refactoring. Snow and Grass type maps complete, desert and fire type maps still need work.

13/03/19

Maps need to be resized from 32x32 to 16x16 and file paths need to be corrected for importing. All quests are nearly complete. Combat still needs to be finalised. Our own A* Search will be implemented for pathfinding. Different dialogue for NPCs needs to be created.

15/03/19

Finalised maps complete, desert, fire, grass, snow however need an additional border. Pathfinding is working but still needs to be optimised. Testing classes for game need to be worked through and need to do testing on the game searching for bugs.
