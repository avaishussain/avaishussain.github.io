The Blog

This is the game we have been working on for the past few months.
It is something we've put a lot of energy into, effort and commitment to have this complete.
We hope you find the game enjoyable and satisifying.

__Development Cycle


Initial GUI

Here we created the first GUI for the game interface as a basic way of navigating.

Upgraded GUI

Over time we upgraded the intitial GUI and made it more gamely and more appealing.

First World tests

We design an intitial basic world to test the designs and how things would be designed in this interface.

Upgraded Worlds

We upgraded the worst designs after we had figured out collision detection. once we had collision mastered we could design a new world.

New exterior

we created a better amount of trees and map designs to fill up the screen and make it a better design to play on.

New mapping

We designed the interiors of the game and the new maps for the houses.

Old inventory design

This was added as a test to see its layout on screen and to see how things would position. we tested functionality to check i f it all worked well and passed tests before we implemeted the designs.

new inventory design.

We upgraded the inventory after had all the placeholders and functionality working.

Sprites

We created more thoroughly designed sprites to give more detail to the game to match the world and the backgrounds better.

Mini Map

We decided to add a minimap to the game to show the user where they are as the world is huge and can get confusing.

__Meetings

22/02/19

Day after the meeting with Antonio in regard to the feedback we got for the quality assurance plan, took the feedback he gave and discussed it. Our idea of white box testing wasn’t exactly correct so when game testing is implemented, we need to revisit. Library is being moved from Slick2d to LIBGDX. New Design on maps and more maps need to be made. Also discussed the story line, if we should leave the ending as a cliff-hanger but decided not to. Amount of Boss enemies per map should be 1.

27/02/19

GUI Cleaned up and made separate classes for each render. Performance has gone up by a lot since the last meeting as the switch to LIBGDX was successful. Working on Saving and loading, still needs a lot of work. Work on different animations for characters.

1/03/19

Combat Animation discussed, need to work on getting the merchants working, purchasing weapons from shops and adding them to inventory. Few bugs found in system which need to be fixed, Memory usage on pause, Player spawns over object layer, Projectile animation is overly aggressive.

6/03/19

Different Theme maps are needed, Snow, Grass, Fire, Dessert etc and plus the rooms for different styled maps. Give enemies attributes, hp, damage etc. issue with projectile and animation fixed and updated into the master. Should make different types of quests. Pathfinding still a major issue.

08/03/19

New Quest Type completed, Character animation completed too, game tracks NPCs and Enemies. Tidied up code, Sprite manager for player animations. Trading between NPCs and player needs to be implemented. Game Screen class needs refactoring. Snow and Grass type maps complete, desert and fire type maps still need work.

13/03/19

Maps need to be resized from 32x32 to 16x16 and file paths need to be corrected for importing. All quests are nearly complete. Combat still needs to be finalised. Our own A* Search will be implemented for pathfinding. Different dialogue for NPCs needs to be created.

15/03/19

Finalised maps complete, desert, fire, grass, snow however need an additional border. Pathfinding is working but still needs to be optimised. Testing classes for game need to be worked through and need to do testing on the game searching for bugs.

16/03/2019

This week sees all the final implementations with every feature being polished off to result in our end product. The team has really pulled their weight this week and we are happy to say we’re proud of our progress.

19/03/2019

Everyone fianlly got together to ensure we had everything working for the game demo, we made sure that game is working. we commmitted all teh changes of the different developers and we ensured that the latest files were merged and functioning.
